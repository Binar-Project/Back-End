# Project Setup

## Step 1: Clone the repo
```bash
git clone https://github.com/Binar-Project/Back-End.git
```
## Step 2 : Open Folder
```bash
cd .\Back-End\
```
## Step 3 : Install Dependencies
```bash
npm install
```
## Step 4 : Run
```bash
npm start
```
## Step 5 : Open 
```bash
localhost:5000
```
